# GoroGrid Energy Monitoring App (Frontend)

## Descripción

La **aplicación de monitoreo de energía GoroGrid** es una solución innovadora para la optimización del consumo energético en edificios inteligentes. El frontend de la aplicación está diseñado para ofrecer una experiencia interactiva y fácil de usar, permitiendo a los usuarios visualizar en tiempo real el consumo energético de sus edificios.

Este repositorio contiene la interfaz de usuario (UI) de la aplicación, desarrollada utilizando **React** y **TypeScript**, lo que permite una integración eficiente con el backend y una interfaz dinámica y responsiva.

## Características

- **Dashboard en tiempo real**: Muestra el consumo de energía de los edificios en tiempo real.
- **Interactividad**: Los usuarios pueden interactuar con gráficos y visualizaciones para obtener información detallada.
- **Responsividad**: La aplicación se adapta a diferentes tamaños de pantalla, proporcionando una excelente experiencia tanto en dispositivos móviles como de escritorio.
- **Optimización de recursos**: Utiliza tecnologías eficientes para asegurar un rendimiento rápido y fluido.

## Estructura del Proyecto

El proyecto se organiza en las siguientes carpetas y archivos:

- **.github/workflows**: Archivos de configuración para integración continua y despliegue automático.
- **node_modules**: Dependencias del proyecto.
- **src**: Contiene todo el código fuente del frontend. Aquí se encuentran los componentes clave de la aplicación, como los gráficos, el dashboard, y otros elementos interactivos.
- **README.md**: Documento con información importante sobre el proyecto.
- **index.html**: El archivo HTML principal de la aplicación.
- **package-lock.json**: Bloqueo de las versiones exactas de las dependencias.
- **package.json**: Definición de las dependencias y scripts de ejecución.
- **vite.config.ts**: Configuración de Vite para la construcción del proyecto.

## Requisitos

Para ejecutar este proyecto, necesitas tener instalado:

- **Node.js** (version 16 o superior)
- **npm** (gestor de paquetes de Node.js)
- **Vite** (para el desarrollo y la construcción)
